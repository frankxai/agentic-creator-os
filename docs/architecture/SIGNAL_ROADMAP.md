# Signal — Road to Top Voice (long-term product development)

The path from "we have a skill pack" to "Frank is a LinkedIn Top Voice, and this is a sellable product." Sequenced so each stage compounds — mirrors the [L99 Blueprint](L99_BLUEPRINT.md) phases, applied to LinkedIn.

---

## Stage 0 — Operate (this week, no build)

**Goal: publish daily, by hand-in-loop, starting now.**
- Surface: **Claude cockpit + Notion board + ACOS Signal pack.** No new repo, no dashboard.
- Fill personas, seed Dream100, connect Firecrawl + Exa.
- Run the [Daily Loop](../../skills/signal/DAILY_LOOP.md): 1 post + 5–10 Dream100 comments/day.
- Log outcomes to Notion (the manual stand-in for the store).
- **This alone starts the flywheel.** Everything below makes it compound harder and eventually sell.

## Stage 1 — Measure (weeks 2–4)

**Goal: the loop learns.**
- Build the Success Oracle (Blueprint Phase 0): grade each post from real metrics + your verdict.
- Wire `learn` to a durable store so `plan` recommends *your* proven hooks/times/formats.
- The compounding claim becomes true here. Top-Voice trajectory becomes measurable, not hoped-for.

## Stage 2 — Scale the intake (weeks 4–8)

**Goal: never run dry, never off-brand.**
- Full research swarm across all source lanes; auto-vetted daily brief.
- Carousel + infographic production at volume via multimodal + canvas.
- Content index of your full back-catalog so nothing is wasted.

## Stage 3 — The Studio (weeks 8–12)

**Goal: a real product surface.**
- Next.js + v0 dashboard on Vercel: calendar → draft → approve → posted → performance.
- Supabase state spine (the `content` table) so drafts/analytics persist and the board is live.
- Typefully official-API integration for one-click human posting + scheduling.
- This is the first thing that could be *sold* (template / hosted Studio).

## Stage 4 — Productize (quarter 2)

**Goal: it works for others, not just you — you're the case study.**
- Package as the **Signal starter template** (one-time) + hosted **Studio** (subscription).
- Two-brand support productized (personal + company) — a natural upsell.
- Done-with-you tier via the existing Workshop OS.
- Open-core: the skills stay MIT in ACOS; the Studio + packs are the product.

---

## What "Top Voice" actually requires (the honest version)

No tool makes you a Top Voice. The tool removes every excuse not to do the two things that do:
1. **Show up daily with a genuine, specific point of view** — Signal removes the blank page and the production tax, so consistency stops being hard.
2. **Be present in your field's conversations** — the Dream100 engagement loop, done by hand-in-loop, is the highest-leverage growth motion and it's fully ToS-safe.

Signal's job is to make daily excellence effortless enough that you actually sustain it, and to learn what works for *you* so every week beats the last. That compounding — sustained 6–12 months — is the path. There is no shortcut, and anyone selling one is selling automation that gets accounts banned.
