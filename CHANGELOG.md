# Changelog

All notable changes to Agentic Creator OS. Restarted at v12.0.0 — earlier
history (v4–v11) was renumbered without releases and is preserved only in git.

## [12.0.0] — 2026-07-12

The open-core reset. Theme: **make every claim true**.

### Fixed
- Skill auto-activation now actually fires: the UserPromptSubmit hook read a
  rules file at a path that never existed and expected a different schema than
  the rules ship in. Both fixed; verified end-to-end.
- CI was red on every push (empty `schedule:` key) — workflow rewritten and
  green, now with real gates.
- Author identity corrected to Frank Riemer (package metadata shipped an
  invented surname).
- OpenCode config pointed at another machine's absolute paths.

### Added
- `LICENSE` (MIT) — previously claimed in four places, shipped nowhere.
- `.claude-plugin/marketplace.json` + component paths in `plugin.json`:
  `claude plugin marketplace add frankxai/agentic-creator-os` now works.
- `hooks/hooks.json` wiring 4 cross-platform, telemetry-only Node hooks.
- `CREATOR.md` identity contract; agents no longer assume any one creator.
- `STATS.md` generated by `scripts/generate-stats.mjs`; CI fails on stale
  counts, empty skills, phantom activation rules, and leaked private content.
- `docs/ARCHITECTURE.md`, `docs/ROADMAP.md`, `docs/VISION.md`.
- DCO sign-off requirement in CONTRIBUTING.md.

### Removed
- Vendored claude-flow trees (~160 files of agents/commands), vendored gstack
  (229 files), gsd system, Anthropic-proprietary document skills (license),
  placebo echo-hooks, broken lib-less hook scripts, dead slash-command shell
  scripts, 5 empty skills, phantom registries (skill-rules-v11.json).
- Personal content from core (first pass): FrankX commands/agents/skills →
  `instances/frankx/`, session telemetry (21 files), employer-internal path,
  internal pricing strategy lines, third-party commit attribution.

### Changed
- `hooks/` now means lifecycle hooks only; the marketing hook-writing KB moved
  to `content-hooks/`.
- All docs rewritten against generated stats; versions unified at 12.0.0.
