## Research-Backed Hooks

### Hook R001: The Focus Study
**Category:** Research
**Target Emotion:** Recognition
**Best Platform:** YouTube, LinkedIn

- **Visual Hook:** Split screen: distracted multitasker vs. single-task deep worker
- **Audio Hook:** Notification sounds chaos → silence → focused typing
- **Text Hook:** "Cal Newport's research: Deep work produces 4x output."

**Why It Works:** Combines credible source with specific multiplier that challenges common behavior.

**Reusable Prompt:**
```
Create a research-backed hook about [TOPIC] that cites a specific study or researcher and includes a concrete metric or finding.
```

---

### Hook R002: The Pattern Analysis
**Category:** Research
**Target Emotion:** Curiosity
**Best Platform:** Twitter/X, TikTok

- **Visual Hook:** Rapid montage of different creators working, then freeze frame highlighting one common element
- **Audio Hook:** Upbeat analysis music, then pause, then "Found it."
- **Text Hook:** "After studying 1,000 creators, one pattern appeared in 94%."

**Why It Works:** Large sample size implies rigor; specific percentage creates curiosity gap.

**Reusable Prompt:**
```
Create a hook that presents analysis of a large group and teases one surprising common pattern.
```

## Counter-Intuitive Hooks

### Hook CI001: The Disproven Rule
**Category:** Counter-Intuitive
**Target Emotion:** Curiosity
**Best Platform:** TikTok, YouTube Shorts

- **Visual Hook:** "10,000 Hours" text appearing then cracking/shattering
- **Audio Hook:** Confident statement, then record scratch, then "Actually..."

### Hook CI002: The Paradox
**Category:** Counter-Intuitive

- **Audio Hook:** Stress sounds releasing, then creative flow music
