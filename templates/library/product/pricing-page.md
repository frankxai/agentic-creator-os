## Pricing Psychology Tips

### Anchoring
- Show annual savings
- Display "most popular" badge
- Use strikethrough for discounts
- Show value vs. cost

### Simplicity
- 3-4 tiers maximum
- Clear feature differences
- Simple language
- Easy comparison

### Urgency
- Limited-time offers
- Trial countdown
- Seasonal discounts
- Early bird pricing

### Value
- Feature benefits, not just features
- ROI calculators
- Cost per user/usage
- Comparison to competitors

## A/B Testing Ideas

### Test These Elements
- Number of tiers (3 vs. 4)
- Monthly vs. annual default
- CTA button text
- Tier names
- Feature presentation
- Price display format
- Trial length
- Guarantee prominence

## Performance Metrics

### Track These
- Conversion rate by tier
- Time on page
- Scroll depth
- CTA click rate
- Trial signup rate
- Upgrade rate
- Churn by tier

### Optimization
- Simplify if confusion
- Highlight value over features
- Test different price points
- Improve feature descriptions
- Add more social proof
