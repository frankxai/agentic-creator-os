## SEO Optimization Checklist

Before publishing, ensure:

- [ ] Primary keyword in title (preferably at the beginning)
- [ ] Primary keyword in first paragraph
- [ ] Primary keyword in at least one H2 heading
- [ ] Meta description includes primary keyword and compelling CTA
- [ ] URL slug includes primary keyword
- [ ] Alt text for all images includes relevant keywords
- [ ] Internal links to 3-5 related articles
- [ ] External links to 2-3 authoritative sources
- [ ] Content length: 2,500-4,000 words minimum
- [ ] Readability score: Grade 8-10 level
- [ ] Mobile-friendly formatting
- [ ] Fast page load time (<3 seconds)
- [ ] Schema markup for article
- [ ] Social sharing meta tags (Open Graph, Twitter Cards)

## Distribution Strategy

### On Publication Day:
1. Share on all social media platforms with platform-specific copy
2. Send to email list with personalized introduction
3. Post in relevant communities (with value-add, not just links)
4. Reach out to mentioned experts/brands for shares

### Week 1:
1. Create social media content series from key points
2. Engage with all comments and shares
3. Monitor for backlink opportunities
4. Update with any new insights from community feedback

### Ongoing:
1. Update quarterly with new data, examples, or insights
2. Repurpose into other formats (video, podcast, infographic)
3. Use as lead magnet or course foundation
4. Build internal link structure as you create related content
