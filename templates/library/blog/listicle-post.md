## Listicle Optimization Checklist

Before publishing:

### Content Quality
- [ ] Each item has clear value and actionable steps
- [ ] Examples and data support each point
- [ ] No fluff or filler items
- [ ] Consistent structure across all items
- [ ] Strong opening and closing

### Formatting
- [ ] Scannable with clear headings
- [ ] Visual elements (images, icons, emojis) for each item
- [ ] Comparison table included
- [ ] Bullet points and numbered lists used effectively
- [ ] White space for easy reading

### SEO
- [ ] Number in title (odd numbers perform better)
- [ ] Primary keyword in title and first paragraph
- [ ] Alt text for all images
- [ ] Internal links to related content
- [ ] Meta description optimized
- [ ] URL slug includes keyword and number

### Engagement
- [ ] Personalized recommendations section
- [ ] Clear action plan
- [ ] FAQ section addresses common questions
- [ ] Social sharing buttons prominent
- [ ] Comment prompt at end

### Viral Potential
- [ ] Title is curiosity-inducing
- [ ] Number is specific (not rounded)
- [ ] Content delivers on promise
- [ ] Shareable quotes or statistics
- [ ] Visual assets for social sharing
