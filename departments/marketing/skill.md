# Marketing Department Skill

## Description

Use this skill when the user needs help with marketing, SEO, analytics, or growth strategies. This includes SEO optimization, content marketing, social media strategy, and performance analysis.

## Triggers

- "Improve my SEO..."
- "Run an SEO audit..."
- "Help me with social media marketing..."
- "Analyze my website traffic..."
- "Create a marketing campaign..."
- "Research keywords for..."
- "Optimize my conversion rate..."
- "Set up analytics for..."

## Capabilities

- Technical SEO auditing and optimization
- Keyword research and analysis
- Content marketing strategy
- Social media marketing
- Analytics setup and analysis
- Conversion rate optimization
- Link building strategy
- Competitor analysis

## Usage Examples

### Example 1: SEO Audit
```
User: "Run a complete SEO audit of my website"

When: User wants to understand SEO issues
Action: Use Marketing Department to crawl, analyze, and report on SEO
```

### Example 2: Keyword Research
```
User: "Find profitable keywords for my niche"

When: User needs keyword research
Action: Use Marketing Department to research and prioritize keywords
```

### Example 3: Marketing Strategy
```
User: "Create a marketing plan for my product launch"

When: User needs marketing strategy
Action: Use Marketing Department to develop comprehensive marketing plan
```

## Workflows

- `seo-audit` - Comprehensive SEO audit workflow
- `keyword-research` - Keyword research workflow
- `marketing-campaign` - Campaign creation workflow

## Related Tools

- `browser-mcp` - Site crawling and analysis
- `filesystem-mcp` - Report storage
- `database-mcp` - Analytics data
- `website-mcp` - SEO implementation

## Configuration

```yaml
marketing:
  seo_priority: high
  track_rankings: true
  include_competitors: true

```
