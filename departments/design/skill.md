# Design Department Skill

## Description

Use this skill when the user needs help with visual design, branding, UI/UX, or creative assets. This includes website design, logo creation, marketing materials, and design system management.

## Triggers

- "Design a logo for..."
- "Create a landing page design..."
- "Help me with branding..."
- "Design marketing materials..."
- "Create social media graphics..."
- "Design an email template..."
- "Improve my website UI..."
- "Create a style guide..."

## Capabilities

- Logo and brand identity design
- Website UI/UX design
- Marketing collateral design
- Social media graphics
- Email template design
- Design system management
- Visual asset organization

## Usage Examples

### Example 1: Logo Design
```
User: "Create a modern logo for my tech startup"

When: User needs a new logo
Action: Use Design Department to generate logo concepts and specifications
```

### Example 2: Landing Page
```
User: "Design a high-converting landing page for my product"

When: User needs landing page design
Action: Use Design Department to create layout and visual specifications
```

### Example 3: Brand Guide
```
User: "Create a brand style guide for my company"

When: User needs brand guidelines
Action: Use Design Department to document brand colors, typography, and usage
```

## Workflows

- `logo-creation` - Logo design workflow
- `landing-page-design` - Landing page workflow
- `brand-guide-creation` - Brand guidelines workflow

## Related Tools

- `filesystem-mcp` - Asset storage
- `website-mcp` - Web design implementation
- `browser-mcp` - Design research
- `database-mcp` - Asset management
